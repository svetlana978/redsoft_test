# Person Service

gRPC сервис для управления данными о людях с автоматическим обогащением через внешние API.

## Запуск

1. Поднять бд: ```make docker-prepare```
2. Запуск миграций для создания таблицы и добавления seed-данных: ```make migrate-up```
Откат миграции и удаление всех данных: ```make migrate-down```
3. Поднять grpc сервис: ```make docker-up-service```
Остановить и grpc сервис, и бд: ```make docker-stop```
Остановить сервис и удалить все данные бд: ```make docker-down-v```
4. Запуск тестов: ```go test ./...``` или ```make test```
